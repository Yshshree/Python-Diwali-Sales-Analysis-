# Python-Diwali-Sales-Analysis-
Diwali Sales Data Analysis using Python (Pandas, NumPy, Matplotlib &amp; Seaborn) to perform Exploratory Data Analysis (EDA), visualize customer purchasing behavior, and generate business insights from festive retail sales data.

**📌 Project Overview**

This project performs Exploratory Data Analysis (EDA) on a real-world Diwali Sales Dataset using Python.

The goal of this analysis is to understand:

Customer purchasing behavior
Sales trends
High-performing product categories
Demographic insights
Business decision patterns

The analysis helps businesses improve marketing strategies, customer targeting, and sales optimization during festive seasons.

**📊 Dataset Information**:

The dataset contains customer transaction details collected during Diwali sales.

**📈 Exploratory Data Analysis (EDA)**

**👩 Gender Analysis:**

**Observations:**

1.Majority buyers are Females.

2.Female customers contribute higher total sales amount.


**👥 Age Group Analysis**

**Findings:**

1.Highest purchasing age group: 26–35 years.

2.Female customers dominate this segment.

**🌍 State-wise Analysis**

**Top Performing States:**

1.Uttar Pradesh

2.Maharashtra

3.Karnataka

These states generated:

1.Maximum Orders.

2.Highest Revenue.


**💍 Marital Status Analysis**

**Key Insight:**

Married women show higher purchasing power compared to others.

**💼 Occupation Analysis**

Top contributing professions:

1.IT Sector

2.Healthcare

3.Aviation

These occupations produced maximum sales revenue.

**🛍️ Product Category Analysis**

Most purchased categories:

1.Food

2.Clothing

3.Electronics

These categories dominate festive shopping trends.

**⭐ Top Selling Products**

Analysis identified top 10 products based on total orders using groupby aggregation and visualization.

**🔍 Key Business Insights**

✅ Married women aged 26–35 are the primary buyers.

✅ Customers from UP, Maharashtra, Karnataka generate maximum revenue.

✅ High-income professional sectors drive sales.

✅ Food, Clothing, and Electronics dominate festive purchases.


**🧠 Business Recommendations**

Target female customers with personalized campaigns.

Focus marketing on 26–35 age group.

Increase inventory for top-selling categories.

Run regional promotions in high-performing states.

Launch festive combo offers.
